# To Run Docker with Docker Compose
- Run Windows PowerShell as Administrator
- Enter the command 'cd' (without quotes) and paste the full path to the project root folder
- Press Enter on your keyboard
- Enter the command 'docker compose up --build' (without quotes)
- Press Enter on your keyboard

DO NOT ATTEMPT TO COPY PROJECT PATH FROM INTELLIJ -- DOESN'T WORK.

# To Check Status of Container in Docker Desktop
- Open Docker Desktop
- Select "Containers" on the sidebar
- Next to the container, click the dropdown arrow to expand details
